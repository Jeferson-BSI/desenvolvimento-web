Missão da W3C

A World Wide Web Consortium (W3C) é uma comunidade internacional que desenvolve e mantém padrões e diretrizes para a World Wide Web. Seu objetivo é garantir o crescimento a longo prazo e a interoperabilidade da Web.

Padrões Web

Um dos aspectos-chave do trabalho da W3C é definir e publicar especificações técnicas e padrões para várias tecnologias web. Esses padrões ajudam a garantir que o conteúdo e as aplicações web funcionem consistentemente em diferentes navegadores e plataformas. Exemplos proeminentes de padrões da W3C incluem HTML, CSS e o Document Object Model (DOM).

Acessibilidade

A W3C coloca um forte foco na acessibilidade na web, com o objetivo de tornar a web utilizável por todas as pessoas, incluindo aquelas com deficiências. As Diretrizes de Acessibilidade para Conteúdo Web (WCAG) são um conjunto de diretrizes desenvolvidas pela W3C para promover a acessibilidade na web.
