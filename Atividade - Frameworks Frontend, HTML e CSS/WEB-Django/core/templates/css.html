Introdução ao CSS e Bootstrap

O que é CSS?

O CSS (Cascading Style Sheets) é uma linguagem de estilo usada para controlar a aparência e o layout de elementos HTML em uma página web. Com CSS, você pode definir cores, fontes, margens, bordas e muito mais.

Como o Bootstrap ajuda com o CSS?

O Bootstrap é um framework front-end que fornece um conjunto de estilos CSS pré-projetados e componentes. Ele simplifica o processo de estilização, permitindo que você use classes CSS do Bootstrap em seu HTML para aplicar estilos prontos.
